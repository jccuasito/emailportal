Paano Gamitin / How to Use
Payslip Status Checker — Employee Guide
1
I-type ang iyong personal na email / Type your personal email

Ilagay ang iyong email address sa kahon, hal. johndoe@gmail.com

2
Suriin ang status / Check the status

Kapag lumabas ang Verified — okay na! Tugma ang email mo sa system.

3
I-click ang "View My Payslips"

Kapag Verified ang status mo, i-click ang button para makita ang listahan ng iyong mga payslip.

4
Tingnan ang buong payslip / View full payslip

Sa Payslip Summary, i-click ang View sa tabi ng date period.

5
I-screenshot / Take a screenshot

Pinakamadaling paraan para mapanatili ang kopya ay ang mag-screenshot ng iyong payslip.

6
Kapag Bounce Email

Kapag Bounce Email ang lumabas, kausapin ang iyong inspector agad para ma-update ang email.

7
Gumamit ng maliliit na letra / Use lowercase

Laging lowercase. Hal. johndoe@gmail.com hindi JohnDoe@Gmail.com
